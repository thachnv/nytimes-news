- Redux for data flow
- Jest for test
- Font Awesome for icons

### How to run the app ?
Make sure you have node installed.

Install yarn (if you don't have yarn yet)

    npm install yarn -g

Install dependencies: 

    yarn install
    
Start development:

    yarn start
    
Run test:
    
    yarn test
    
Build production:

    yarn build
    
Or just access the deployed app here:

    https://nytimes-thachnv.herokuapp.com/
